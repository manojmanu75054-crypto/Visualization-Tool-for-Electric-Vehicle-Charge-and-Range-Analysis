# Visualization Tool for Electric Vehicle Charge and Range Analysis

## Overview

The **Visualization Tool for Electric Vehicle Charge and Range Analysis** is an interactive Tableau dashboard developed to analyze electric vehicle specifications and performance. The dashboard helps users compare electric vehicles based on driving range, charging capability, price, efficiency, acceleration, and other important features through interactive visualizations.

---

## Objectives

- Analyze electric vehicle specifications.
- Compare vehicle range across different brands.
- Study the relationship between vehicle price and driving range.
- Analyze rapid charging availability.
- Compare electric vehicle performance.
- Develop an interactive dashboard using Tableau.

## Key Insights

- Electric vehicles offer different driving ranges depending on the manufacturer.
- Premium electric vehicles generally provide higher range and better performance.
- Most vehicles in the dataset support rapid charging.
- Vehicle efficiency varies across different brands.
- Interactive filters allow users to compare vehicle specifications easily.
